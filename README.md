相比第一版自己的代码技术相对更成熟。结构更清晰。

## Build Setup
```
# 先把项目克隆到本地
git clone https://github.com/martin-yin/dogeyeMovie.git

# cd 到dogeyeMovie/v2
cd dogeyeMovie 

# 安装包依赖  建议各位老哥使用淘宝镜像 因为比较快！！！
npm install

# 启动端口
npm run serve
```
